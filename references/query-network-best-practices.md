## Core Concepts
- **Query Network:** Clustered user searches by proximity, lexical relations, intent layers.
- **Gap Analysis:** Information Gain = Unique EAV not covered by competitors.

## Best Practices
1. Cluster Types: Lexical (synonyms), Proximity (semantic distance), Intent (informational vs transactional).
2. Tools: Advertools for SERP scraping, SentenceTransformers for similarity scoring.
3. Gap Scoring: High gain = Missing EAV with high search volume.
4. Always negate low-value (e.g., -filter:replies).

## Examples
- Query: "Best Electric Scooters Pakistan" → Network: "Price in PKR", "Top Models 2026", "Where to Buy in Peshawar".
- Gaps: If competitors miss "Battery Life Comparison" → Priority High.
