## Core Philosophy
- **Core Section:** Money-making attributes, direct EAV, central user intent (high commercial value).
- **Outer Section:** Historical/support signals, bridge topics for momentum/depth (low direct traffic but boosts authority).

## Real-World Examples (From Koray Case Studies)
1. **Niche: Electric Scooters Pakistan**
   - Core: Types (foldable, off-road), Features (battery life, speed), Best Models 2026, Buying Guide.
   - Outer: History of E-Scooters in Pakistan, Regulations, Maintenance Tips, User Stories from Peshawar.

2. **Niche: Digital Marketing Agency Karachi**
   - Core: Services (SEO, PPC, Social Media), Pricing, Case Studies, How to Choose Agency.
   - Outer: Evolution of Digital Marketing in Pakistan, Google Updates History, Tools Reviews.

3. **Niche: Visa Applications Germany**
   - Core: Types (Student, Work, Tourist), Requirements, Application Process, Fees.
   - Outer: Germany Immigration History, Neighboring EU Countries, Success Stories.

## Best Practices
- Ratio: 60% Core / 40% Outer for most niches.
- Always connect Outer to Core via internal links (exact title anchors).
- Use for gap filling: If competitor weak on Outer, prioritize there for quick wins.
