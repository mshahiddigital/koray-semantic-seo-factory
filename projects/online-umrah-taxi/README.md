# Online Umrah Taxi — Project Workspace

This folder contains a complete (offline) Koray-style semantic SEO workflow pack for the niche:
**"Online Umrah Taxi"** (Hajj/Umrah transport across Makkah, Madinah, Jeddah, Taif, Riyadh).

## Files
- `topical-map.md`: site architecture + page inventory
- `network/query-network.md`: clustered query universe + intent mapping
- `network/internal-linking-plan.md`: hub/spoke link graph + rules
- `briefs/*.md`: ready-to-write semantic briefs
- `audit/hcu-quality-audit.md`: quality & trust checklist
- `scripts/`: optional inputs for Python tooling

## Notes
- Any numeric claims (prices, distances, travel times) are intentionally written as **placeholders** so you can insert verified local data.
