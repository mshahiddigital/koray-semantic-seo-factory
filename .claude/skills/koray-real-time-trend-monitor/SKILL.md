# Koray Real-Time Trend Monitor

## Workflow
1. Confirm the niche + country/language + seed entities.
2. Collect signals (choose what’s available):
   - GSC/analytics deltas (best)
   - Competitor SERP observations (queries, titles, FAQ shifts)
   - Web research when the user explicitly wants recent data
3. Extract actionable deltas:
   - New intents/questions
   - Emerging entities/attributes
   - Policy/regulation changes
   - Competitor information gain (new sections/pages)
4. Map each delta to actions:
   - New node/page
   - Section additions
   - Internal linking updates
   - Brief updates (EAV/lexical)

## Output
- **Trend summary** (bullets)
- **Delta table**: `Signal | Evidence | Recommended action | Priority`
- **Updates to topical map / briefs / network**
